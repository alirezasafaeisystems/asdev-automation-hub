# Branding Contract
Version: 1.0

## Required Product Touchpoints
- Public admin interface footer must contain attribution:
  - `Built by Alireza Safaei (ASDEV)`
- Public brand page route must exist:
  - `/brand`

## SEO Baseline (for public routes)
- Brand page title and description must be explicit.
- Canonical path must be present for brand route.
- External profile link must use `target="_blank"` with `rel="noopener noreferrer"`.

## Acceptance
- Footer attribution visible on `/`.
- `/brand` is reachable without authentication.
- Branding and ownership statements are consistent with repository README.
